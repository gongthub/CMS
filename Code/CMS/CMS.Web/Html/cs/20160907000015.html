新闻资讯

　9月2号晚上，武警湖北总队二支队官兵在巡逻时成功处置一起诈骗行为暴露、实施抢劫事件，路边的监控摄像头记录下了事件的全过程。

　　晚上8点左右，位于武汉市武昌区省政府门前的洪山路和东一路交汇处，当时这里的灯光比较昏暗，路上的行人也比较少。

　　当巡逻小组巡逻至省政府门前时，战士王友昌忽然看到一名穿黑色背心的男子神色慌张向东侧逃窜，有两名群众在后方追赶并大喊“抢劫了，抢劫了”。巡逻组组长谭维随机命令小组成员兵分两路，一路迂回包抄，一路尾随追击，对犯罪分子实施包围。
