家用中央空调价格相比普通空调贵还是便宜？

一提到中央空调，很多人首先联想到的就是一些特别豪华的酒店饭店里面用的，价格一定很贵。其实不然，中央空调现在已经逐步进入了千家万户的大门， 那么家用中央空调价格比起普通空调来说是贵还是便宜呢？

对于家用中央空调价格比较，要从两个方面来说，第一点是设备的初始造价，普通空调一间屋子需要装一个室内机在对应一台室外机，如果有三间卧式一个客厅一个餐厅那么就需要装五个室外机；

而要使用家用中央空调则只需要安装一个室外机即可带起室内的五台室内机，所以从设备造价来说还是家用中央空调价格更低一些。

其次再从设备运行费用来说，普通空调五台内机运行需要开启五台外机，这样五台设备分开耗电运行成本极大。如果使用家用中央空调，开启五台室内机是需要运行一台室外机， 只需要一台室外机的电量消耗，运行成本减少了很多。

那么有人就要问了，如果我使用家用中央空调，只有一个房间需要开空调，也就是是需要开启一台室外机，那么这么大的外机耗电岂不是很大？中港海通小编来告诉您，如果您只开启一台室内机， 那么室外机也只会有一小部分功率，消耗一台室内机的电量。

从初始造价和运行成本两个方面来说，家用中央空调价格是相对普通空调要低很多的，而且更节能。
